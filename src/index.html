User

Sign Up

Post Endpont : /user/signup

Parameter Type Required
name Name of the user Yes
email Email of the User. Yes
passord Set a password for the User Account. Yes
age Age of the user *not required. No
mob User's mobile number
Only indian number for example:+911234567890		 No

Log in

Post Endpont : /user/login

Parameter Type Required
email Email of the user's account. Yes
Password Password of the user's account. Yes

User methods

Login is required for all

Request type Endpont Details
(Get) /user/me To get user accounts details.
(Patch) /user/me Make a patch request to update user's data, like name, age, password, mobile number.
(Post) /user/logout To log out from current device.
(Post) /user/logout/all To log out from all the devices.
(Delete) /user/me Make a delete request to delete user.

Tasks

Login is required for all request.

Request type Endpont Details
(Post) /task To add new task.
Parameter Type Required
title sting Yes
completed boolean No, default value = false
(Patch) /task/[task-id] Make a patch request to update task data, like title, completed.
(Get) /task/[task-id] To get a single task.
(Get) /task To get all the task created by a single user.
(Delete) /task/[task-id] To delete a single task.
